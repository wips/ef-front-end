ef-front-end
==

Front end part for [ef]
---------------------------

Set up:

1. Install nodejs
2. Run nmp install -g grunt-cli
3. Run npm install -g bower
4. Run npm install
5. Run bower install

Local server:

1. Run grunt server
2. Open [localhost]

Tests:

1. Run grunt test or open [tests]

[ef]:https://github.com/wips/ef/
[localhost]:http://localhost:9000/
[tests]:http://localhost:9000/test/
